Dados: e suas funções

A principal mecanica do RPG, os dados servem dentro desse mundo para definir a facilidadade(ou dificuldade na maioria das vezes) de uma tarefa que o "player" deseja que seu personagem faça. O sistema de dados inclui 7 desses, os quais tem número de faces distintas e que servem para determinar resultados para diferentes tarefas dentro do jogo.

Funções dentro do jogo

D20

O principal dado de dnd, é com ele que construimos nossas aventuras e ações, é com ele que por exemplo rolamos os testes de furtividade no caso de um ladino.

D4

Um dado diferente dos comuns, ele geralmente é utilizado para danos menores, como adagas, machadinhas ou socos. Reza a lenda que apenas Tehlu conseguiu montar uma torre de dados começando pelo D4.

D6

O dado classico, todos ja tiveram um D6, ele é utilizado para causar danos medios alem de em algumas ocasiões servir de debuf. Como exemplo temos o debuf de deslocamento quando você machuca uma perna.

D8

Um dos dados menos usados, ele pode ser usado de dado de dano ou de vida dependedo da sua classe. Geralmente é o dado que todo mundo zoa por "só servir para fazer torres de dado".

D10

Um D10 é um ótimo dado para quem gosta de personagens brutos, ele é utilizado como dano alto de uma arma comum, ele serve de dado de vida para as classes mais fortes e resistentes.

D12

Se vc acha que o d10 é um dado bruto, o D12 é ainda mais. uma unica classe utiliza este dado como dado de vida, os barbaros, ja que eles sao muitos brutos e fortes. o D12 tem poucas funções mas ainda sim, tem um formato bonitinho.
